# 🎓 RollCall+

RollCall+ is a student attendance and academic companion app built with React Native, Expo, Node.js, and Cloudflare.

The app helps students track attendance, monitor GPA, view analytics, receive notifications, and gain insights into their academic performance through a modern and user-friendly interface.

## 🚧 Project Status

Active Development – New features and improvements are continuously being added.

## ✨ Features

* 📊 Attendance Dashboard
* 🎯 Attendance Predictor
* 📚 SGPA & CGPA Tracking
* 📈 Attendance Analytics
* 🔔 Smart Notifications

* 📱 Clean & Responsive UI
